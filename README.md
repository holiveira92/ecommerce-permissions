# Permission Management System
> System to manage ecommerce permissions


## Requeriments

* PHP 8.2
* Laravel 10
* Mysql 8
* Node 20
* VueJS 3


## Install

1 - Run composer
```sh
 composer install
```

2 - Configure .env
```sh
 configure database access
```

3 - Execute migrations
```sh
 php artisan migrate
```


## Local Execution
APP_PORT=80

Laravel Sail
```sh
 sail up -d
```

Vite
```sh
 npm run dev
```

ou 

Docker
```sh
 docker-compose up -d
```
